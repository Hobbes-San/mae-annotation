      • createIDTracker

        private java.util.Hashtable<java.lang.String,AttID> createIDTracker()
        The IDTracker hashtable keeps one ID for each element that has an ID, and increments the number so that no two tags of the same type will have the same ID.
        Returns:
        a Hashtable containing a key for each element in the mDtd paired with an AttID that contains the information about the IDs for that element (prefix, name, etc).
        See Also:
        AttID

      • createIDsExist

        private HashCollection<java.lang.String,java.lang.String> createIDsExist()
        mIdsExist is a Hash collection used only when filling the database from a file. It holds the IDs that have already been read in, and is used to ensure that no duplicate element-id pairs are entered. This used to be done by checking the database, but this system is faster and allows the number of batch executions to be lowered.
        Returns:
        a HashCollection that will contain all IDs that are in use for each tag in the DTD
